# CutCutCut

Application for transforming one link to another.

Deployed on https://cutcutcut.herokuapp.com

## Motivation

CutCutCut will help you to do your links:

* Shorter;
* Anonymous;
* Semantic.

## Click stats

Also CutCutCut save stats about every click on link:

* Time;
* IP;
* Browser;
* System.

And finally build charts about dynamics of click.

## Private links

Default links is a public. Everyone can see stats of it. If you want to make links private you need to sign in with application account. Private links will be avalible on "Tools -> Private" tab.
